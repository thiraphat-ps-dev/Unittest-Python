# Unittest-Python

## Installation
Download and install [Node.Js](https://nodejs.org/en/download/) for install package.
<br/>

Install Python3

```bash
$ npm install python
```

Check Python version

```bash
$ python -V
```

Result

```bash
Python 2.7.16
```

Use command python filename.py to run test
```bash
$ python filename.py
```
